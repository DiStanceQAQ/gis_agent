# GIS Agent 开发任务清单（LLM Plan-and-Execute）

更新时间：2026-04-03

这份清单只保留“当前目标”任务：  
把系统落成 **LLM 主导的 Plan-and-Execute GIS Agent**，并将后端编排统一迁移到 **LangGraph**。

状态约定：

- `[x]` 已完成
- `[~]` 进行中
- `[ ]` 待完成

---

## 0. 当前目标定义

1. 任务解析、规划、推荐都由 LLM 生成结构化结果。  
2. 执行由 LangGraph 工具图驱动，并沉淀可观测事件与步骤详情。  
3. 输出处理后数据下载 / 方法说明 / 摘要，支持追问重跑。  
4. 前端是 GIS 工作台，不是聊天气泡产品。
