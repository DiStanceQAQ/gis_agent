## 0. 当前目标定义（Backend Only）

1. 任务默认进入审批态，批准前不执行。  
2. 用户可拒绝执行，任务进入 `cancelled`。  
3. 每个计划步骤都通过 ReAct + Function Calling 执行。  
4. `OperationPlan` 非空且合法，禁止空计划成功。  
5. 全链路审计可按 `task_id` 查询。
